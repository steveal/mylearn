			static 是默认值。任意 position: static; 的元素不会被特殊的定位。一个 static 元素表示它不会被“positioned”，一个 position 属性被设置为其他值的元素表示它会被“positioned”。
		
    	relative 表现的和static一样，除非你添加了一些额外的属性
    	
    	在一个相对定位（position属性的值为relative）的元素上设置 top 、 right 、 bottom 和 
    	left 属性会使其偏离其正常位置。其他的元素则不会调整位置来弥补它偏离后剩下的空隙。
